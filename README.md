# [datapool](https://www.npmjs.com/org/datapool)
🗄️ 一些常用的数据CDN池子，主要是地图数据等。

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/hai2007/datapool/issues)，欢迎参与维护！
